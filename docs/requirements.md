# 수진프레시 요구사항

> 냉장고 재료 관리 + 장보기 + 식비 가계부 + 커플 일정 공유 통합 웹앱. 모바일 퍼스트.

---

## ✅ 완성된 기능

### 재료 관리 (🥬 재료 탭)
- [x] 재료 추가: 이름, 양(숫자+단위: 개/g/kg/ml/L/기타), 카테고리(냉장/냉동/팬트리), 날짜, 메모
  - 날짜는 **유통기한** 또는 **개봉일자** 중 선택
  - 날짜 라벨 클릭 → 달력 자동 열림
- [x] 카테고리 필터: **전체 / 냉장 / 냉동 / 팬트리** (각 칩에 개수 표시)
  - 상단 로고 클릭 → 자동으로 "전체" 모드로 이동
- [x] 정렬 옵션: **⏰ 임박순 / 📅 오래된순 / ✋ 내 순서**
- [x] 내 순서: **편집 모드 토글** 버튼으로 드래그 아이콘 켜고/끄기
- [x] 재료 카드 탭 → 수정 다이얼로그 (전체 필드)
  - ChatGPT 레시피 추천 링크 포함
- [x] 소비 처리 (soft delete — `consumed_at` 타임스탬프)
- [x] **소진 기록** (🍴 FAB): 사용한 양 입력 → `usages` 테이블에 로그
  - 재료 단위와 같은 계열(무게/부피)이면 **남은 양 자동 차감**
  - L↔ml / kg↔g 단위 변환 지원
  - 남은 양이 0 이하면 **자동 소진 처리**

### 구매 필요 (🛒 구매 탭)
- [x] 항목 추가: 이름만 빠르게 (상단 인라인 입력)
- [x] **🎤 음성인식 추가** (Web Speech API): "우유 두 개" 같은 자연어 → 자동 파싱·추가
  - 연속 입력 지원 (마이크 한 번 켜면 여러 개 연속)
  - 단위 자동 인식 (개/팩/병/봉지/kg/g/ml/L/리터 등 다양)
- [x] 장봤음 체크 토글 (optimistic UI + 250ms 디바운스)
- [x] 항목 탭 → 수정 다이얼로그
- [x] × 버튼 즉시 삭제 + **5초 실행 취소 토스트**
- [x] **자동 숨김**: 체크된 지 24시간 지나면 목록에서 제외 (DB는 유지)

### 식비 가계부 (💰 가계부 탭)
- [x] 월별 합계: ◀ ▶ 이동으로 이전 달 조회
- [x] 카테고리별 비율 바: 식자재 / 외식 / 배달 / 음주 / 카페·간식 (5종)
- [x] 날짜별 그룹 목록 (오늘/어제 강조, 날짜별 소계)
- [x] 지출 기록 다이얼로그: 설명, 금액(원), 카테고리, 날짜, 메모
- [x] **구매처 선택** (식자재 카테고리에서만 노출): 마트 / 편의점 / 시장 / 기타(자유 입력)
- [x] 항목 탭 → 수정 / × → 삭제 + 실행 취소

### 커플 일정 (🗓️ 일정 탭)
- [x] 일정 추가: 제목, 날짜, 시간(10분 단위 또는 🌙 종일), 주인(🩷 수진 / 💙 종빈 / 💛 SJB), 메모
- [x] **기간 일정** (📆 체크): 시작일 + 종료일 — 여러 날에 걸친 여행·출장·기간 기념일 (자동 종일)
- [x] 주간 리스트 뷰: 날짜별 그룹 (오늘/내일/어제 강조) + 오름차순
- [x] 카드 왼쪽 세로 색띠로 주인 구분 (분홍 / 하늘 / 노랑)
- [x] 지난 일정은 하단 접기/펼치기 섹션으로 분리
- [x] 카드 탭 → 수정·삭제 다이얼로그
- [x] 외부 캘린더(구글/네이버) 연동 없음 — 앱 자체 DB에만 저장

### 공통 UI
- [x] 상단 탭: 🥬 재료 / 🛒 구매 / 💰 가계부 / 🗓️ 일정 (4탭 세그먼트)
- [x] **SJB 로고 헤더** (투명 PNG, 다크모드 자동 반전)
- [x] **주아체(Jua) 전체 폰트** (Google Fonts 자동 로드)
- [x] 모든 다이얼로그: Esc 키로 닫기, 백드롭 클릭 닫기
- [x] **메모 입력란 자동 확장** — 내용 길이에 따라 높이 자동 조절 (스크롤 불필요)
- [x] 서버 액션 실패 시 빨간 에러 박스로 메시지 표시
- [x] 한글 폰트 폴백 체인 (Apple SD Gothic Neo, Malgun Gothic)
- [x] 다크모드 자동 대응

## 🚫 명시적 비범위 (안 할 것)

- 로그인/인증 (현재는 단일 기기·단일 URL 기준. 일정 "주인"은 라벨로만 구분)
- 재료 바코드 스캔
- 레시피 추천 알고리즘 (ChatGPT 링크로 대체)
- 유통기한 OCR 인식
- 외부 캘린더(Google/Naver) 연동
- 일정 반복(매년 기념일 자동 반복) — 수동 입력으로 갈음

---

## 🧭 설계 원칙

- **절대 hard delete 없음.** 모든 "삭제"는 `consumed_at` 또는 `deleted_at` 타임스탬프 기록.
- **모바일 퍼스트** — PC는 그 다음.
- **로그인 없음** — 일정 "주인"은 라벨(`me`/`partner`/`both`)로만 구분, 같은 URL을 둘이 공유해 쓰는 전제.
- **Server Component + Server Action** 조합. 별도 API 레이어 없음.
- **Optimistic UI + fire-and-forget** 패턴으로 네트워크 지연에 UI가 끌려가지 않게.
